# Issue: bash: pip: command not found

## 상황: bash: pip: command not found

## 생각해낸 방안:

- pip3 사용

## 방안: pip3 사용
맥북이라면 pip 대신 pip 3 명령으로 설치를 해보시는 것을 권해드립니다. 맥북에는 기본적으로 파이썬2가 깔려 있어서 pip 명령으로 하시면 파이썬2용 라이브러리가 설치된답니다.
