# Issue: intellij 검색 대상 폴더 exclude

## 상황: dist라던가 필요없는 폴더도 intellij에서 검색돼서 제외

<br/>

## 알게된 부분 정리:

- 프로젝트에서 exclude

<br/>

## 개념: 프로젝트에서 exclude

<br/>
  
  ```
  File > Project Structure > Modules > Sources > Mark as Excluded (red icon)
  ```
