# Issue: linux 서버에서 메모리 사용률 올라갈 때 처리

## 상황: 메모리 사용률 100프로를 쳐서 해당 부분 확인 후 처리해야하는 상황

## 생각해낸 방안:

- 서버 접속 후 top 명령어로 메모리를 사용하는 프로세스 확인
- shift m 눌러서 메모리 순으로 정렬
- 해당하는 Pid를 kill -9 pid로 죽인다.
